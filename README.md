# connectome-fidelity
Representational geometry as a fidelity metric for connectome-constrained neural emulations. Uses RSA to compare population codes across pretrained Flyvis ensemble models vs. random baselines. Built on [Lappalainen et al. 2024 (Nature)](https://www.nature.com/articles/s41586-024-07939-3). Toward a principled fidelity framework for whole-brain emulation.
